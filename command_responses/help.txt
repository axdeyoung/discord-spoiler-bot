Available Commands:

[help, h] <command>
    Print this help text.
    If a command is specified, specialized help is given for that command

listmedia, l
    Print a list of available media to block spoilers for

unspoil [media]
    Mark yourself as avoiding spoilers for the specified media

spoil [media]
    Unmark yourself as avoiding spoilers for the specified media
    
addmedia [media]
    Adds a role for the specified media
    Requires role-editing permission

deletemedia [media]
    Deletes the specified media role
    Requires role-editing permission
